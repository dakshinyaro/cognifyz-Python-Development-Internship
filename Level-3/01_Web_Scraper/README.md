# 🌐 Web Scraper

## 📌 Overview

The Web Scraper is a Python application that downloads a webpage, extracts all H2 headings using BeautifulSoup, displays them in the console, and saves them to a text file.

---

## ✨ Features

- Downloads webpages
- Extracts H2 headings
- Saves output to a text file
- Handles connection errors
- Beginner-friendly implementation

---

## 🛠️ Technologies Used

- Python 3
- Requests
- BeautifulSoup4

---
