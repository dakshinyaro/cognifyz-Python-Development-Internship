# 📊 Data Visualization

## 📌 Overview

This project reads sales data from a CSV file and visualizes it using a line chart with Matplotlib.

---

## ✨ Features

- Reads CSV data
- Creates a line graph
- Displays monthly sales
- Beginner-friendly code
- Handles missing file errors

---

## 🛠️ Technologies Used

- Python 3
- Matplotlib
- CSV Module

---

## 📁 Project Structure

```
02_Data_Visualization/

```

## ▶️ Installation

