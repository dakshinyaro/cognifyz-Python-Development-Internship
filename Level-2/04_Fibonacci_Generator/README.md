# 🔢 Fibonacci Generator

## 📌 Overview

The Fibonacci Generator is a Python console application that generates the Fibonacci sequence for a user-specified number of terms. The sequence begins with 0 and 1, and each subsequent number is the sum of the previous two numbers.

---

## ✨ Features

- Generates Fibonacci sequence
- User chooses the number of terms
- Handles invalid input
- Allows multiple executions
- Beginner-friendly implementation
