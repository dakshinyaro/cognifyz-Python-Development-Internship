# 🤖 Number Guesser

## 📌 Overview

The Number Guesser is a Python console application where the computer guesses the number the user is thinking of. It uses the Binary Search algorithm to efficiently find the correct number.

---

## ✨ Features

- Computer guesses the user's number
- Uses Binary Search
- Interactive gameplay
- Fast and efficient
- Beginner-friendly code

---

## 🛠️ Technologies Used

- Python 3
