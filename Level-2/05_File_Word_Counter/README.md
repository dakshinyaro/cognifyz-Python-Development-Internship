# 📄 File Word Counter

## 📌 Overview

The File Word Counter is a Python application that reads a text file and displays the total number of lines, words, and characters. It demonstrates file handling and text processing using Python.

---

## ✨ Features

- Reads text files
- Counts lines
- Counts words
- Counts characters
- Handles missing files
- Allows multiple analyses

---

## 🛠️ Technologies Used

- Python 3
