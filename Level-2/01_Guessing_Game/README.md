# 🎮 Guessing Game

## 📌 Overview

The Guessing Game is a Python console application where the computer randomly selects a number between 1 and 100. The player keeps guessing until the correct number is found. The program provides hints after every guess and counts the number of attempts.

---

## ✨ Features

- Random number generation
- Hints for higher or lower guesses
- Counts the number of attempts
- Input validation
- Play multiple rounds
- Beginner-friendly code

---

## 🛠️ Technologies Used

- Python 3
- Random Module
