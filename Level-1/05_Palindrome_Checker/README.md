# 🔄 Palindrome Checker

## 📌 Overview

The Palindrome Checker is a Python console application that checks whether a given word or sentence is a palindrome. It ignores spaces and letter case, making the validation more flexible.

---

## ✨ Features

- Checks words and sentences
- Ignores uppercase and lowercase letters
- Ignores spaces
- Allows multiple checks
- Beginner-friendly code

---

## 🛠️ Technologies Used
