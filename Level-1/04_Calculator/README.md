# 🧮 Calculator

## 📌 Overview

This project is a menu-driven calculator developed using Python. It performs the four basic arithmetic operations and includes input validation and error handling.

---

## ✨ Features

- Addition
- Subtraction
- Multiplication
- Division
- Prevents division by zero
- Handles invalid numeric input
- Menu-driven interface
- Multiple calculations without restarting

---

## 🛠️ Technologies Used

- Python 3
