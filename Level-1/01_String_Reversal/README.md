# String Reversal

## Description
This Python program takes a string from the user and displays the reversed version of the string.

## Features
- Accepts user input
- Reverses any string
- Displays both original and reversed strings
- Simple and beginner-friendly

## Technologies Used
- Python 3

## How to Run

```bash
python string_reversal.py
```

## Example Output

```
========== String Reversal Program ==========

Enter a string: hello

Original String : hello
Reversed String : olleh
```

## Concepts Used

- Functions
- Strings
- String Slicing
- User Input
