# 📧 Email Validator

## 📌 Overview

The Email Validator is a Python console application that checks whether an email address is valid using Regular Expressions (Regex). It provides instant feedback and allows users to validate multiple email addresses without restarting the application.

---

## ✨ Features

- Validates email addresses using Regular Expressions
- User-friendly console interface
- Supports multiple email checks in one session
- Displays clear success or error messages
- Beginner-friendly and easy to understand

---

## 🛠️ Technologies Used

- Python 3
- Regular Expressions (`re` module)
