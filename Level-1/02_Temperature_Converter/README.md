Temperature Converter

📌 Description

The Temperature Converter is a simple Python program that converts temperature values between Celsius, Fahrenheit, and Kelvin. It provides a menu-driven interface where the user selects the type of conversion and enters the temperature value.

---

🚀 Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Convert Celsius to Kelvin
- Convert Kelvin to Celsius
- User-friendly menu
- Displays results with two decimal places
- Beginner-friendly code using functions

---

🛠️ Technologies Used

- Python 3
